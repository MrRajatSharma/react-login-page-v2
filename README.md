# react-login-page-v2
---
## Setup
1. run `yarn install` 
2. run `npm start` 
3. open `http://localhost:4001/`

## Setup Social login
1. Replace `<YOUR_GOOGLE_CLIENT_ID>` with Google Client ID
2. Replace `<YOUR_FB_APP_ID>` with FB App ID
